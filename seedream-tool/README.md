# BytePlus Seedream Tool Plugin

## Overview
This plugin provides HTTP tool access to BytePlus Seedream image generation APIs.

## Version
0.0.1

## Setup
1. Install the plugin from Dify Marketplace or upload the local .difypkg file.
2. Add credentials for the provider.
3. Use the Seedream tool in your Dify applications.

## Configuration
- API Key
- API Endpoint Host
- SSL Verify
- CA Bundle Path (optional)

## Usage
Provide a prompt and model ID, then call the tool to generate images. The tool returns the raw JSON response, including the generated image URL or base64 data.
