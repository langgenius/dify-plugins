# Privacy Policy

## Data Collection

This plugin sends the following data to the Dinq API (https://api.dinq.me):
- Platform type (GitHub, LinkedIn, or Google Scholar)
- Query string (username, profile URL, or researcher name)

## Data Usage

The data is used solely for the purpose of analyzing the requested profile and generating insights.

## Data Storage

No personal data is stored beyond what is necessary for the analysis request. Cached results may be stored temporarily to improve performance.

## Third-Party Services

This plugin uses the Dinq API service. For more information about Dinq's privacy practices, please visit https://dinq.me/privacy.

## Contact

For privacy-related inquiries, please contact: support@dinq.me
