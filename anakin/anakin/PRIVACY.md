# Privacy Policy

**Anakin Dify Plugin**
**Last Updated:** February 2025

## Overview

This privacy policy describes how the Anakin plugin for Dify handles user data when integrated into your Dify applications.

## Data Collection

### Data We Collect

The Anakin plugin collects and processes the following data:

| Data Type | Purpose | Stored |
|-----------|---------|--------|
| API Key | Authentication with Anakin API | Stored securely by Dify |
| URLs | Submitted for scraping | Temporarily processed |
| Search Queries | Submitted for search | Temporarily processed |
| Scraped Content | Returned to your workflow | Not stored by plugin |

### Data We Do NOT Collect

- Personal user information
- Usage analytics or tracking data
- Conversation history
- User credentials (beyond API key)

## Data Processing

### How Data is Processed

1. **API Key**: Stored securely within Dify's credential management system. Never logged or transmitted to third parties.

2. **URLs and Queries**: Sent directly to Anakin's API servers for processing. Processed in real-time and not stored beyond the request lifecycle.

3. **Scraped Content**: Retrieved from target websites and returned to your Dify workflow. The plugin does not store or cache this content.

## Third-Party Services

This plugin connects to the following third-party service:

### Anakin API

- **Provider:** Anakin (anakin.io)
- **Data Sent:** URLs, search queries, scraper configurations
- **Data Received:** Scraped content, search results, research reports
- **Privacy Policy:** [https://anakin.io/privacy](https://anakin.io/privacy)

## Data Security

- All communication with Anakin API uses HTTPS encryption
- API keys are transmitted securely using the `X-API-Key` header
- No sensitive data is logged by the plugin

## Data Retention

- The plugin does not retain any user data
- All data processing is transient and stateless
- Scraped content exists only for the duration of the API request

## User Rights

Users have the right to:

- **Access**: View what data is being sent to the Anakin API
- **Delete**: Remove their API key from Dify at any time
- **Opt-out**: Stop using the plugin to cease all data processing

## Children's Privacy

This plugin is not intended for use by children under 13 years of age. We do not knowingly collect data from children.

## Changes to This Policy

We may update this privacy policy from time to time. Changes will be reflected in the "Last Updated" date above.
