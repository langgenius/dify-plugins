# Privacy Policy

Last updated: July 20, 2026

This plugin sends the configured Adanos API key and the parameters required for
the selected tool to `https://api.adanos.org`. Parameters can include a stock
ticker or crypto symbol, a selected data source, a result limit, and optional
date boundaries.

The plugin does not collect, sell, persist, or log API keys, tool parameters, or
API responses. Dify and Adanos may process requests according to their own
deployment and service policies. Refer to the
[Adanos privacy policy](https://adanos.org/privacy) for the hosted API.

The plugin calls only the fixed Adanos HTTPS API and does not execute code,
write files, place trades, or contact user-provided URLs.

Questions can be sent to support@adanos.org.
