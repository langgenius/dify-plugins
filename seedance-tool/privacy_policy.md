# BytePlus Seedance Tool Plugin Privacy Policy

## Data Collection and Use
This plugin only processes data required to call BytePlus Seedance APIs, including:
- Provider credentials (API Key, endpoint).
- User prompts and optional image inputs.

The plugin transmits the above data to BytePlus ModelArk for inference and does not store it locally after the request completes.

## Data Sharing
Data is shared with BytePlus ModelArk as the service provider. Please review BytePlus ModelArk data processing policies:
https://docs.byteplus.com/en/docs/modelark/BytePlus_ModelArk_Data_Processing

## Data Retention
The plugin itself does not retain user data. Data retention is governed by Dify and BytePlus ModelArk.
