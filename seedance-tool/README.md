# BytePlus Seedance Tool Plugin

## Overview
This plugin provides HTTP tool access to BytePlus Seedance video generation APIs.

## Version
0.0.1

## Setup
1. Install the plugin from Dify Marketplace or upload the local .difypkg file.
2. Add credentials for the provider.
3. Use the Seedance tools in your Dify applications.

## Configuration
- API Key
- API Endpoint Host
- SSL Verify
- CA Bundle Path (optional)

## Usage
Create a video generation task with a prompt and optional images, then query the task status to retrieve the video URL.
