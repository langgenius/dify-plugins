# Privacy Policy

## Overview
The Olostep Dify plugin connects Dify agents to the Olostep web data API 
(api.olostep.com). This document describes what data is transmitted when 
using this plugin.

## Data Transmitted
When using this plugin, the following data is sent to the Olostep API:
- URLs provided by the user for scraping, crawling, or mapping
- Search queries provided by the user
- Research tasks or questions provided by the user

This data is transmitted solely to fulfill the requested operation and is 
subject to Olostep's own privacy policy.

## Data Collection by This Plugin
This plugin does not collect, store, log, or process any user data itself. 
No personal data is retained by this plugin.

## Third Party Services
This plugin relies on the Olostep API. By using this plugin you agree to 
Olostep's Terms of Service and Privacy Policy:
- Privacy Policy: https://www.olostep.com/privacy-policy
- Terms of Service: https://www.olostep.com/terms

## Credentials
Your Olostep API key is stored securely by Dify and is only used to 
authenticate requests to the Olostep API. It is never logged or shared.

## Contact
For privacy-related questions contact: info@olostep.com
