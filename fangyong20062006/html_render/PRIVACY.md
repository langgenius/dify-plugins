# Privacy Policy

## Data Collection
This plugin does **not** collect, store, or transmit any personal data or user content.

## Data Processing
All HTML strings provided to this plugin are processed locally within the Dify runtime environment. The plugin wraps the input HTML into a self-contained page and returns it. No content is sent to any external server or third-party service by the plugin.

## Note on User-Provided HTML
The HTML content provided by the user may itself contain references to external resources (such as CDN-hosted scripts or images). Such references are part of the user's own content; the plugin does not add, fetch, or proxy any external resources on its own.

## Data Retention
This plugin does not retain any input or output data after a request completes.
