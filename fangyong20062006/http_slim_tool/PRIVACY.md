# Privacy Policy

## Data Collection
This plugin does not collect or store any personal data.

## Network Access
The paginated-fetch tool sends HTTP/HTTPS GET requests to the URL explicitly provided by the user, in order to retrieve data for processing. This network access to user-specified endpoints is a core function of the tool. The field-slimming and summary tools operate entirely locally on the data passed to them. The plugin does not send any data to third-party services.

## Data Processing
Fetched or supplied data is slimmed, paginated, or summarized within the Dify runtime and returned to the workflow. No data is retained after the request completes.

## User Responsibility
The user is responsible for ensuring they have the right to access the target endpoints.
