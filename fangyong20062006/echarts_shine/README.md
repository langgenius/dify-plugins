# ECharts Shine Charts

Render publication-ready ECharts visualizations in the elegant Shine theme for Dify.

## Features

- 30+ chart types: bar, line, pie, scatter, radar, heatmap, funnel, gauge, candlestick, sankey, tree/treemap, sunburst, parallel, boxplot, graph, themeriver, calendar, and 3D/GL charts
- Built-in China national and provincial geo-maps
- All ECharts assets embedded for fully offline rendering (no external CDN)

## Usage

Add the chart tool you need (e.g. **Bar Chart**, **Line Chart**) to your Dify workflow, pass in the data, and receive a rendered chart.

## Privacy

All rendering runs locally within the Dify runtime. No data is collected or transmitted. See `PRIVACY.md` for details.
