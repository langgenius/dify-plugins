# Privacy Policy

## Data Collection
This plugin does **not** collect, store, or transmit any personal data or user content.

## Data Processing
All chart data provided to this plugin is rendered locally within the Dify runtime environment. All ECharts JavaScript assets and geo-map data are embedded in the plugin package; rendering requires no network access and no external CDN.

## Data Retention
This plugin does not retain any input data or generated charts after a request completes.

## Third-Party Services
This plugin does not use any third-party services or external APIs. It bundles the open-source ECharts library (Apache-2.0) locally.
