# Privacy Policy

## Data Collection
This plugin does **not** collect or transmit any personal data to external services.

## Data Processing & Storage
All key-value data provided to this plugin is stored locally within the Dify plugin container (in-process and persisted to local disk). The cache runs entirely in-process with no real Redis server or external connection. No data is sent to any external server or third-party service.

## Data Retention
Cached data persists locally until it expires (TTL), is explicitly deleted, or the underlying storage is cleared. No data leaves the Dify environment.

## Third-Party Services
This plugin does not use any third-party services or external APIs.
