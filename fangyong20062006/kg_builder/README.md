# Knowledge Graph Builder

Turn Markdown documents into knowledge-graph-style chunks for Dify knowledge bases.

## Features

- Each chunk carries entity and relation annotations to boost multi-hop reasoning and retrieval recall
- Built-in validator checks entity coverage, relation completeness, and trap blocks
- Outputs a quality report

## Usage

Add the **Knowledge Graph Builder** tool to your Dify workflow, pass in the Markdown text, and receive graph-style chunks. Use the validator tool to check chunk quality.

## Privacy

All processing runs locally within the Dify runtime. No data is collected or transmitted. See `PRIVACY.md` for details.
