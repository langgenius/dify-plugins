# Privacy Policy

## Data Collection
This plugin does **not** collect, store, or transmit any personal data or user content.

## Data Processing
All document text provided to this plugin is processed locally within the Dify runtime environment. Entity and relation extraction, knowledge-graph chunking, and validation all run in-process. No content is sent to any external server or third-party service.

## Data Retention
This plugin does not retain any input or output data after a request completes.

## Third-Party Services
This plugin does not use any third-party services or external APIs.
