# Privacy Policy

## Data Collection
This plugin does **not** collect, store, or transmit any personal data or user content.

## Data Processing
All Markdown text provided to this plugin is processed locally within the Dify runtime environment. The cleaning pipeline (noise removal, sentence repair, terminology normalization, deduplication, etc.) runs entirely in-process. No content is sent to any external server or third-party service.

## Data Retention
This plugin does not retain any input or output data after a request completes. Generated files exist only transiently within the Dify environment.

## Third-Party Services
This plugin does not use any third-party services or external APIs.
