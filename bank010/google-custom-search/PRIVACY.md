# Google Custom Search Plugin Privacy Policy

## 数据收集和使用

本插件使用 Google Custom Search API 进行网络搜索。在搜索过程中：

1. 搜索查询会被发送到 Google 服务器
2. 搜索结果由 Google 返回
3. 插件不会存储任何搜索历史或用户数据

## 数据安全

- API 密钥和搜索引擎 ID 作为敏感信息存储
- 所有数据传输使用 HTTPS 加密
- 不收集或存储用户个人信息

## 第三方服务

本插件依赖以下第三方服务：
- Google Custom Search API
- Google Cloud Platform

## 数据保留

- 不存储任何用户数据
- 搜索结果仅在当前会话中临时使用
