## FlexMD-to-Word

**Author:** chuhaojiqi  
**Version:** 0.0.1  
**Type:** tool

### Description (EN)
- Convert Markdown into polished Word (.docx) with flexible style control.
- Supports headings, lists, tables, images, code blocks, spacing before/after.
- Zero external credentials; validates runtime dependencies (markdown, html2docx).
- Built for Dify workflows; python runner 3.12.

### Usage
- Required input: `markdown` (string, the content to convert).
- Optional inputs: `filename` (default `document.docx`), `style_profile` (e.g., 学术论文/商务报告/技术文档), plus font/size overrides.
- Output: a Word `.docx` file.

### Notes
- Icon: `_assets/icon.svg` / `_assets/icon-dark.svg`.
- No OAuth/credentials needed.
