# Privacy Policy — Kronvex Memory Plugin

## Data Collection

This plugin does **not** collect any personal data directly.

When you use this plugin, the text content you provide (memories, queries) is forwarded to the Kronvex API (`https://api.kronvex.io`) for storage and semantic search. No email addresses, IP addresses, or personal identifiers are collected or stored by the plugin itself.

## Data Storage

Memory content is stored on Kronvex servers hosted in the European Union (EU). Data is associated with your API key and agent ID only.

## Data Sharing

No data is shared with third parties. Data is processed solely by the Kronvex API to provide the memory service.

## GDPR Compliance

Kronvex is fully GDPR-compliant and EU-hosted. For full details, see the [Kronvex Privacy Policy](https://kronvex.io/privacy).
