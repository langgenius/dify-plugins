# AIhubMix 插件隐私政策

欢迎使用 AIhubMix 插件（以下简称 "插件"）。本隐私政策旨在明确插件收集、使用、存储、保护及共享用户个人信息的相关规则，你使用本插件即表示同意遵循本政策所述的数据处理方式。

## 一、收集的个人数据类型

### 1. 直接识别信息
- **账户注册信息**：电子邮件地址、用户名、密码（密码经加密处理存储）
- **支付相关信息**：加密货币支付场景下的钱包地址（完整支付卡详情由第三方支付处理商单独处理）

### 2. 间接识别信息
- **设备与网络信息**：IP 地址、设备相关标识
- **使用行为信息**：API 调用记录、模型使用情况、Token 消耗量、关联 Web 应用程序 URL
- **技术存储信息**：通过 cookies 等技术存储的设备偏好信息

### 3. 组合识别信息
本插件不收集此类可单独或结合其他信息识别个人的组合数据（如年龄、性别、职业等）。

### 4. 第三方服务数据收集说明
插件使用以下第三方服务，其数据收集规则遵循对应服务的隐私政策：
- **云服务提供商**：Microsoft Azure（隐私政策链接：https://privacy.microsoft.com/zh-cn/privacystatement）
- **支付处理商**：遵循其官方隐私政策（具体以实际合作支付服务商公示为准）
- **分析服务**：Google Analytics（隐私政策链接：https://policies.google.com/privacy）

## 二、数据使用目的

- **提供核心服务**：处理 API 请求，实现大模型（LLM）访问功能
- **账户管理**：创建、验证及维护用户账户，保障账户使用安全
- **计费结算**：处理支付流程、管理订阅服务、生成及发送账单
- **服务优化**：分析使用模式，提升插件性能、API 响应速度及功能体验
- **客户支持**：响应用户咨询、解决使用过程中遇到的问题
- **安全保障**：检测并防范欺诈、滥用行为及各类安全威胁
- **合规通信**：发送服务通知、功能更新信息（营销类信息需获得用户明确同意）

## 三、数据共享规则

- 不向任何第三方出售用户个人信息
- 仅向以下必要第三方共享数据，且第三方仅可在履行职能范围内使用：
  - **服务支撑类**：Microsoft Azure（云存储）、Google Analytics（使用分析）、第三方支付处理商（支付结算）
  - **合规要求类**：响应法院命令、传票等合法请求，或为保护 AIhubMix、用户及公众合法权益必需的披露
  - **业务转让类**：若涉及合并、收购或资产出售，用户信息可能作为转让资产的一部分转移，受让方需遵守本隐私政策
- 插件不会向 OpenAI 或其他大模型提供商披露 API 调用者的身份信息（包括 IP 地址、用户代理字符串等）

## 四、数据安全与保护

- **传输安全**：所有 API 通信通过 TLS/SSL 加密协议保障数据传输安全
- **存储安全**：敏感数据采用行业标准加密技术存储，服务器及数据库部署于 Microsoft Azure 云服务平台
- **访问控制**：严格限制内部员工对个人数据的访问权限，仅授权人员可在必要范围内接触
- **安全审计**：定期评估并更新安全防护措施，防范数据泄露、丢失或被滥用

## 五、API 调用内容特别说明

- 插件不收集、记录或存储用户通过 API 发送的提示、查询等输入文本，也不存储大模型返回的响应文本
- 不将用户 API 调用内容用于模型训练，也不主动审查 API 调用内容
- 大模型提供商可能存在临时存储提示数据的情况（具体以对应模型提供商的官方数据政策为准，可通过模型页面查询）

## 六、数据保留与存储

- **保留期限**：仅在实现本政策所述目的所需的最短时间内保留个人信息，法律要求或允许的更长保留期除外
- **数据删除**：不再需要的个人信息将通过安全方式删除或匿名化处理
- **存储位置**：用户数据存储于美国 Microsoft Azure 云服务平台，非美国地区用户使用插件即同意数据跨境传输至美国处理

## 七、用户权利与选择

- **信息修改**：可通过插件关联平台的用户界面更新个人可修改信息
- **账户注销**：可通过平台设置页面或联系客服申请删除账户及相关个人数据
- **Cookie 控制**：可通过浏览器设置管理或删除 cookies，禁用部分 cookies 可能影响插件部分功能正常使用

## 八、特殊人群保护

插件不面向 13 岁以下儿童提供服务，不故意收集未成年人个人信息。若发现误收集，将立即删除相关数据。

## 九、政策更新与适用范围

- **政策更新**：本隐私政策可能不时更新，重大变更将在插件关联平台公示，更新日期标注于政策顶部，继续使用插件即表示接受变更后的政策
- **适用法律**：受美国特拉华州法律管辖，相关争议由特拉华州法院专属管辖
- **终止条款**：服务终止后，所有权、免责声明、责任限制等核心条款继续有效

## 十、联系说明

如对本隐私政策有疑问或需行使数据相关权利，可通过插件关联平台的客服渠道联系我们。
