## 🚀 快速开始

### 1. 构建插件
```bash
cd tencent_cloud_search_plugin
./build.sh
```

### 2. 安装到Dify
1. 上传生成的 `.difypkg` 文件到Dify
2. 配置腾讯云API密钥
3. 开始使用

### 3. 基础使用
```json
{
    "query": "人工智能发展趋势"
}
```

## 🔧 支持的参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| query | string | ✅ | 搜索关键词 |
| mode | select | ❌ | 搜索模式（0/1/2） |
| site | string | ❌ | 指定网站 |
| count | number | ❌ | 结果数量（尊享版） |
| from_time | number | ❌ | 开始时间戳 |
| to_time | number | ❌ | 结束时间戳 |
| industry | select | ❌ | 行业筛选（尊享版） |
