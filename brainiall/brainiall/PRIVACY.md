## Privacy

This plugin does not collect, store, or process any personal data on its own. All data is sent directly to the Brainiall LLM Gateway API endpoint, which proxies requests to AWS Bedrock.

For more information about data handling, please refer to:
- Brainiall Privacy Policy: https://brainiall.com/privacy
- AWS Bedrock Privacy: https://aws.amazon.com/bedrock/security-compliance/
